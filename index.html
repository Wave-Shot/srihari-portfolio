AI-Powered Test Case Generator

Streamlit | Gemini API | Image Processing

RAG System: Intelligent Agent & Search

FastAPI | FAISS | Sentence Transformers | Agentic Reasoning

Engineered a sophisticated Retrieval-Augmented Generation (RAG) system for educational content. The system processes and chunks NCERT PDFs into embeddings using Sentence Transformers and indexes them with FAISS for fast semantic search. It's served by FastAPI with an intelligent agent layer capable of selective tool invocation (e.g., Wikipedia search), demonstrating multi-source synthesis and advanced agentic reasoning.

Real-Time Social Media Sentiment Dashboard

Streamlit | BERT | VADER | Reddit/YouTube Data

Developed a real-time sentiment analysis dashboard for dynamic public opinion insights. The system integrates multiple models (BERT, VADER, TextBlob) to analyze live data streams from platforms like Reddit and YouTube. The results are visualized seamlessly within a Streamlit dashboard, providing immediate and actionable decision-making support based on sentiment trends.
